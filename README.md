# 数字のお話 iOS

SwiftUI製の小さな占いアプリです。日付、名前、迷っていること、自由入力の言葉を1から9の数字に変換し、短い読み解きを表示します。

## 入っている機能

- テーマ選択
- 数字の読み解き
- 結果履歴
- 共有
- 効果音と触覚フィードバック
- 広告差し替え用の `AdService`
- プライバシーマニフェスト

## 提出前に必要なこと

1. Macで `LifeRoulette.xcodeproj` を開く
2. Teamを設定する
3. Bundle ID `com.tokyonasu.numberstory` をApple Developer側で作る
4. ReleaseビルドをArchiveし、App Store Connectへアップロードする

この環境はWindowsなので、XcodeのArchiveとTransporterでのビルド提出は実行できません。
